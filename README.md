# IBM-Attrition
IBM Employee Attrition Analysis Report
Objective: Analyze IBM employee attrition data using exploratory data analysis (EDA) to identify trends affecting employee turnover.
Dataset Preparation
- Loaded IBM attrition dataset.
- Renamed inconsistent column names.
- Checked for missing values.
- Removed MaritalStatus column.
- Generated descriptive statistics.
Analysis Performed
- Department, Travel and Education Field distributions.
- Gender and Age count plots.
- Correlation analysis.
- Pairplot for feature relationships.
- Attrition rate grouped by Department, Job Role, Gender and OverTime.
- Bar charts and combined subplot visualization.
Key Insights
• Overtime appears to have a strong association with higher attrition.
• Attrition varies across departments and job roles.
• Gender-based attrition comparison was evaluated.
• Age distribution helps identify workforce demographics.
• Correlation matrix highlights relationships among numerical variables.
Conclusion
The notebook performs a solid exploratory analysis of IBM employee attrition. The results indicate that overtime, department, and job role are important factors related to employee turnover. Further work could include predictive modeling using classification algorithms (Logistic Regression, Random Forest, XGBoost), feature importance analysis, and model evaluation.
